Зворотний проксі - це сервер, який знаходиться перед веб-серверами 
і пересилає запити клієнтів (наприклад, веб-браузера) цим веб-серверам.
Зворотні проксі-сервери зазвичай реалізуються для підвищення безпеки, 
продуктивності та надійності.

У комп’ютерних мережах зворотний проксі - це тип проксі-сервера, 
який отримує ресурси від імені клієнта з одного або кількох серверів.
Потім ці ресурси повертаються клієнту, виглядаючи так, 
ніби вони походять від самого зворотного проксі-сервера.

В основному зворотні проксі використовується для балансування навантаження.

Деякі переваги зворотного проксі:
Балансування навантаження
Захист від атак
Кешування
Шифрування SSL
