запустіть на терміналі команду 

git reflog

яка дасть вам список коммітів з хешами

тоді ви можете перейти до будь-якого комміта, використовуючи його хеш фіксації

git checkout <hash>


Або відмінити історію змін

Перевірте свій рефлог

git reflog

Виберіть комміт, попередній до вашого rebase, і замініть "x" відповідним номером нижче:

Просто скасуйте останнє оновлення:

git reset --hard HEAD@{x}

..

git push -f origin master

Видаліть свій push запит і зробіть новий.
